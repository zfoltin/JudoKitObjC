Overview

  • You can get a copy of the receipt for an individual transaction by creating a Receipt Object and calling .sendWithCompletion(() -> ()) including the receipt ID for the transaction in the path. Alternatively, you can receive a list of all the transactions. By default it will return 10 transactions. These results are returned in time-descending order by default, so it will return the latest 10 transactions.
